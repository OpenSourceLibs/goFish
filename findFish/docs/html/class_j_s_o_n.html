Detailed Description

A JSON builder which takes in values as strings an formats them into valid JSON.

Author
Tomas Rigaux
Date
May 15, 2019

A helper class for dynamically building valid JSON strings out of objects passed in. The class currently does not use templates, and instead relies on the user to convert their objects to strings first before passing them into the object.
